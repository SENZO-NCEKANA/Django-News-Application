Django News Application - Capstone Project

Repository URL: https://github.com/SENZO-NCEKANA/Django-News-Application

Project Description:
A comprehensive Django-based news application featuring:
- Role-based access control (Readers, Journalists, Editors)
- Article management with approval workflow
- Publisher and subscription system
- REST API with full CRUD operations
- Docker containerization
- Comprehensive Sphinx documentation
- Security best practices with proper .gitignore

Technologies Used:
- Django 5.2.5
- Python 3.11
- MySQL/SQLite database
- Docker & Docker Compose
- Sphinx documentation
- REST Framework
- Bootstrap for frontend

Features:
- User authentication and authorization
- Article creation, editing, and approval
- Publisher management
- Newsletter system
- Advanced search and filtering
- Email notifications
- API endpoints for all functionality

The application is production-ready with Docker support,
comprehensive documentation, and security measures in place.
